wikipedia-yz
===========================
A riak benchmark program
wikipedia-yz stores wikipedia contents as objects.

Requirements
------------
- msgpack-rpc
- riak-client

Usage
-----
$ ./wikipedia-yz -s (num of servers) -p (num of workers) (path to xmldump)
